# Dokumentationsstruktur

Diese Ablage trennt aktive V1-Dokumente, Evidenzsammlung und alte Quellen klar voneinander.

## Ordner

- `v1/planning/`: aktive Planungs- und Freigabedokumente
- `v1/handoff/`: Runbooks und manuelle Abnahme fuer die letzten V1-Schritte
- `v1/evidence/`: Zielordner fuer echte Ausfuehrungsnachweise vom Test- oder Produktivhost
- `archive/`: alte oder externe Quelldokumente, die nicht Teil der laufenden Repo-Struktur sind

## Empfohlener Ablauf

1. In `v1/planning/` Status und Freigabekriterien lesen.
2. In `v1/handoff/` die realen Restaufgaben ausfuehren.
3. Ergebnisse in `v1/evidence/` ablegen.
